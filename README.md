# scoop-zig

A scoop bucket for the zig compiler and related tools.

To install zig:

```sh
# add scoop bucket
scoop bucket add scoop-zig https://github.com/enndubyu/scoop-zig

#install zig
scoop install zig
```

## Tools in this bucket:
- **zig** - *zig compiler* 
- **zig-dev** - *zig compiler (development version)*
